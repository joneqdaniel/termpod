*BloodRayne 1* was released in 2002. It was developed by [[Terminal Reality]] and published by Majesco. It is a third-person action game. It is the predecessor to [[BloodRayne 1]].

# Reference

BloodRayne uses the [[POD3 file format|Pod 3 Format Reference]] to store its data.

It furthermore uses following file formats within the POD files:

 * ABL
 * ANI
 * BFM
 * BST
 * CAGE
 * FNT
 * FX
 * JUG
 * LPB
 * LVL
 * MP3
 * SBS
 * SCB
 * SIM
 * SKB
 * SMB
 * TEX
 * TXT
 * VCR
 * WAV
